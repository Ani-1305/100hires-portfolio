# 100Hires Portfolio Project — Setup & Onboarding

## Tools Installed

1. **Cursor IDE** — Downloaded and installed from [cursor.com](https://cursor.com/). Cursor is a VS Code-based code editor with built-in AI capabilities.
2. **Claude Code Extension** — Installed via the Cursor Extensions marketplace (search: "Claude Code"). This extension integrates Anthropic's Claude AI directly into the editor for code assistance.
3. **Codex Extension** — Installed via the Cursor Extensions marketplace (search: "Codex"). This adds OpenAI Codex-powered features for code generation and completion.

## Steps Completed

1. Downloaded and installed Cursor IDE on my machine.
2. Opened Cursor and navigated to the Extensions panel (sidebar icon).
3. Searched for and installed the **Claude Code** extension.
4. Searched for and installed the **Codex** extension.
5. Created a GitHub account and set up this public repository.
6. Cloned the repository locally using Cursor's built-in Git integration (`Ctrl+Shift+P` → "Git: Clone").
7. Created this `README.md` file inside Cursor describing the full setup process.
8. Committed and pushed the changes to GitHub.

## Issues Encountered & How I Solved Them

- **Finding the right extensions:** The Cursor Extensions marketplace has many results for common terms. I made sure to match the exact extension names mentioned in the instructions ("Claude Code" and "Codex") and verified the publisher before installing.
- **Git authentication:** When pushing to GitHub for the first time from Cursor, I was prompted to authenticate. I used GitHub's browser-based sign-in flow that Cursor offers, which handled the token setup automatically.
- **Understanding the Git workflow:** As someone newer to Git, I looked up how `git add`, `git commit`, and `git push` work together. Cursor's Source Control panel (the branch icon in the sidebar) made this easier by providing a visual interface — I staged files with the "+" button, typed a commit message, and clicked "Commit & Push."
